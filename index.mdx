CCVim is a feature-rich Vim clone built for [ComputerCraft](https://computercraft.cc) computers. It brings the power of modal editing, syntax highlighting, multi-file tabs, and a built-in file explorer to your in-game terminal.

## Features

<CardGroup cols={2}>
  <Card title="Modal editing" icon="pen-to-square">
    Insert, append, and normal modes with familiar Vim keybindings. Press `TAB` to return to normal mode.
  </Card>
  <Card title="Syntax highlighting" icon="highlighter">
    Built-in Lua syntax highlighting with an extensible syntax file system for any language.
  </Card>
  <Card title="Multi-file tabs" icon="table-columns">
    Open multiple files simultaneously and switch between them with `gt` and `gT`.
  </Card>
  <Card title="File explorer" icon="folder-open">
    Navigate your filesystem with the built-in CCFXP directory browser.
  </Card>
  <Card title="In-file search" icon="magnifying-glass">
    Search forward and backward through files with `/` and `?`, with wrap-around and search history.
  </Card>
  <Card title="Copy/cut/paste" icon="clipboard">
    Yank lines with `yy`, delete with `dd`, and paste with `p` or `P`.
  </Card>
</CardGroup>

## Quick example

```
vim myfile.lua
```

Open CCVim from the ComputerCraft shell. Press `i` to enter insert mode, type your code, then press `TAB` to return to normal mode. Save with `:w` and quit with `:q`.

<Note>
  CCVim version 0.733 is the current release. It runs on ComputerCraft computers and turtles, and supports external monitor peripherals.
</Note>
