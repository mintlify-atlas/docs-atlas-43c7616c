This guide gets you from a fresh CCVim installation to confidently editing files in under five minutes.

## Open a file

From the ComputerCraft shell, run CCVim with a filename:

```bash
vim myfile.lua
```

If the file does not exist, CCVim creates it when you first save. If you run `vim` with no arguments, you see the welcome screen. Press `:q` then `Enter` to exit it.

## Your first edit

<Steps>
  <Step title="Enter insert mode">
    Press `i` to enter insert mode. The status bar at the bottom shows `-- INSERT --`.
    
    <Note>
      CCVim uses **TAB** to exit insert mode, not Escape. This is the most important difference from standard Vim.
    </Note>
  </Step>
  <Step title="Type your code">
    Type normally. Use the arrow keys or `Backspace` to correct mistakes.
    
    ```lua
    print("Hello from CCVim!")
    ```
  </Step>
  <Step title="Return to normal mode">
    Press `TAB` to exit insert mode and return to normal mode. The `-- INSERT --` indicator disappears.
  </Step>
  <Step title="Save the file">
    Type `:w` then press `Enter`. The status bar confirms the save.
  </Step>
  <Step title="Quit">
    Type `:q` then press `Enter` to exit CCVim.
    
    To save and quit in one command, use `:wq`.
  </Step>
</Steps>

## Essential keybindings

### Moving around

| Key | Action |
|-----|--------|
| `h` | Move left |
| `j` | Move down |
| `k` | Move up |
| `l` | Move right |
| `0` | Beginning of line |
| `$` | End of line |
| `gg` | Go to first line |
| `G` | Go to last line |
| `[N]G` | Go to line N |

### Entering edit modes

| Key | Action |
|-----|--------|
| `i` | Insert before cursor |
| `a` | Append after cursor |
| `I` | Insert at line start |
| `A` | Append at line end |
| `o` | New line below, enter insert |
| `O` | New line above, enter insert |
| `TAB` | Return to normal mode |

### Editing in normal mode

| Key | Action |
|-----|--------|
| `x` | Delete character under cursor |
| `dd` | Delete (cut) current line |
| `yy` | Yank (copy) current line |
| `p` | Paste after cursor |
| `P` | Paste before cursor |

### Ex commands

| Command | Action |
|---------|--------|
| `:w` | Save file |
| `:q` | Quit |
| `:wq` | Save and quit |
| `:q!` | Quit without saving |
| `:e [file]` | Open a file |

## Search in a file

In normal mode, press `/` then type a search term and press `Enter`. CCVim jumps to the first match. Press `n` to jump to the next match, `N` for the previous.

```
/print
```

Press `?` instead of `/` to search backward.
